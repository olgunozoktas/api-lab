Klavyeden çıkmadan yapabileceğiniz her şey. Tüm kısayollar macOS'ta
**⌘** kullanır (API Lab şu an yalnızca macOS'ta yerel olarak çalışıyor).

## İstek gönderme

| Kısayol | Eylem                                                  |
| ------- | ------------------------------------------------------ |
| ⌘ Enter | Aktif isteği gönder                                    |
| ⌘ .     | Devam eden isteği iptal et                             |
| ⌘ S     | Aktif isteği klasöre kaydet                            |
| ⌘ N     | Yeni istek — aktif sekmeyi boşalt                      |
| ⌘ L     | URL çubuğuna odaklan + tümünü seç (tarayıcı standardı) |

## Sekmeler

| Kısayol  | Eylem                                      |
| -------- | ------------------------------------------ |
| ⌘ T      | Yeni sekme                                 |
| ⌘ ⇧ T    | Son kapatılan sekmeyi geri aç              |
| ⌘ W      | Aktif sekmeyi kapat                        |
| ⌘ 1..9   | N. sekmeye atla (N > sekme sayısıysa sona) |
| ⌥ ⌘ →    | Sonraki sekmeye geç                        |
| ⌥ ⌘ ←    | Önceki sekmeye geç                         |
| Orta tık | İmlecin altındaki sekmeyi kapat            |
| Çift tık | Sekmeyi yeniden adlandır                   |
| Sürükle  | Sekme sırasını değiştir                    |

Çok sekme açıkken şerit yatay kayar — trackpad ile swipe edin. Şerit
scrollbar'ı bilerek gizli; ⌘ 1..9 veya quick switcher ile geçtiğinizde
aktif sekme otomatik olarak görüş alanına gelir.

## Gezinme & yerleşim

| Kısayol   | Eylem                                                                                                                      |
| --------- | -------------------------------------------------------------------------------------------------------------------------- |
| ⌘ K / ⌘ P | Quick switcher — kaydedilmiş istekleri fuzzy ara. ⌘ K modern command-palette geleneği; ⌘ P Postman alışkanlığını yansıtır. |
| ⌘ B       | Kenar çubuğunu aç/kapat (kayıtlı istekler + geçmiş)                                                                        |
| ?         | Bu Guide hub'ı aç                                                                                                          |

## Composer

| Kısayol             | Eylem                                                        |
| ------------------- | ------------------------------------------------------------ |
| ⌘ V (URL çubuğunda) | cURL komutu yapıştır — method/URL/header/body otomatik dolar |

## Henüz kısayolu olmayan ipuçları

- **Sağ tık** — çoğu yüzeyde (sekmeler, kenar çubuğu, kayıtlı
  istekler, geçmiş kayıtları, yanıt header'ları) en kullanışlı
  eylemlerin bulunduğu bağlam menüsü vardır.
- **Panel boyutlandırma** — kenar çubuğu / composer / yanıt
  görüntüleyici arasındaki bölücüleri sürükleyin. Çift tık ile
  varsayılana döner.
- **Durum filtresi pilleri** — geçmiş kenar çubuğunun üstündeki
  pillerle listeyi tek durum sınıfına daraltabilirsiniz (2xx / 3xx /
  4xx / 5xx). Az önce 500 aldıysanız izini sürmek için pratiktir.
