Send your first request:

1. Type a URL in the address bar — e.g. `https://api.github.com/users/octocat`.
2. Pick a method (`GET` / `POST` / etc).
3. Press `⌘ Enter` or click **Gönder / Send**.

The response body, headers, and timing land in the right pane. Hit
`⌘ S` to save the request into a collection so you can run it again
later.

## The three panes

- **Sidebar (left)** — collections (saved requests, organized in
  folders) and history (every request you've sent).
- **Composer (middle)** — request editor: URL, params, headers,
  auth, body, scripts, GraphQL.
- **Response (right)** — body, headers, raw, examples.

Resize the dividers by dragging. Double-click a divider to reset.

## What's next

- Press `?` anytime to open this guide.
- Press `⌘ P` for the quick switcher across tabs / collections /
  history.
- Open Settings (top-bar gear) for a full keyboard shortcut list.
