---
title: Git-based collection sync
date: 2026-05-17
---

API Lab can now sync your collections and environments through any
git repo — no cloud account, no proprietary format, opt-in only.

Point **Settings → Collection sync** at an SSH repository URL and hit
**Set up sync**. From then on, every change to a collection or
environment is committed and pushed in the background, and on launch
API Lab pulls the latest first. Auth runs through your system SSH
agent, so no token is ever stored by the app.

If two machines diverge, a conflict banner asks you to keep your
local copy or take the remote — collections are never silently
merged. Sync stays completely off until you set it up.
