Body skeletons + hover descriptions reach the rest of the gallery.
**GitHub**, **OpenAI**, **Slack**, and **Notion** write endpoints now
import with a JSON body already filled in — create an issue, post a
Slack message, generate a chat completion, create a Notion page, all
without leaving the app to look up the payload shape.

**Linear** — GraphQL-only — opens with a starter `viewer` query in the
GraphQL composer, so it runs the moment you add your token.

With Cloudflare and Stripe done earlier, all seven integration
providers now import genuinely ready-to-use collections.
