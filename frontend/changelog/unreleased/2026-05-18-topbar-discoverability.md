The icon-only buttons in the top bar — **Mock**, **Diff**,
**Integrations**, **Guides**, and **Changelog** — were mystery glyphs
with no visible label. Each now shows a proper tooltip on hover, so
the feature behind it is no longer hidden. The Guides tooltip also
shows its **?** shortcut.

The in-app keyboard reference (Guide hub → Keyboard shortcuts) was
missing two bindings that already worked — **⌘ ,** to open Settings
and **⌘ ⇧ E** to open the Environments editor. Both are now listed,
so the reference matches what the app actually does.
