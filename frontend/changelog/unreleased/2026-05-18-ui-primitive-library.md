---
title: A refreshed design-system foundation
date: 2026-05-18
---

API Lab's interface gains a set of shared building blocks — tooltips,
badges, loading skeletons, spinners, and popovers — built once and
properly, so the visual polish landing across the next few releases
stays consistent everywhere.

The first visible touch: the gRPC service browser's loading indicator
is now a cleaner, consistent spinner. The rest of these primitives
quietly power upcoming improvements to notifications, loading states,
and empty screens.
