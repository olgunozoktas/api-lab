---
title: Examples rows now show when each example was saved
date: 2026-05-12
---

Each saved-example row in the response → Examples tab now carries a
small relative-time chip ("3m", "2h", "5d") right before the body
size, so "which example is the newest one I captured?" is visible at
a glance. Hover the chip for the exact save timestamp.

The size text next to it now also picks up the same perceptual-band
colour the response header uses — muted under 100 KB, orange at
100 KB–1 MB, red beyond. Tooltip on the status pill now reads the
full status text (e.g. `404 Not Found`) like the rest of the app.
