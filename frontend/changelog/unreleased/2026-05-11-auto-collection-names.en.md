---
title: Collection items auto-name themselves from the URL too
date: 2026-05-11
---

Saved requests in the sidebar now get the same treatment tabs got
earlier: if the stored name is still the placeholder ("Yeni istek" /
"New request") and the request has a URL, the sidebar shows
`METHOD shortUrl` instead. A folder with several freshly-added
requests no longer reads as three identical "New request" rows.

Double-clicking to rename pre-fills the input with the derived label,
so committing the auto-name is one Enter away. The stored name stays
unchanged until you actually save the rename — your manual edits
always win.
