---
title: Talk to MCP servers
date: 2026-05-20
---

API Lab can now connect to **Model Context Protocol** servers. A new
**MCP servers** button in the top bar opens a panel where you can:

- point at a server over **stdio** — a locally-installed server
  launched as a child process (`npx -y @modelcontextprotocol/...`), or
- point at a remote server over **HTTP**,
- list the tools the server exposes, and
- invoke a tool with JSON arguments and see its result.

stdio servers run through the native bridge — the same mechanism that
makes CORS-free HTTP work — so they're available in the desktop app
and cleanly disabled in browser dev mode.
