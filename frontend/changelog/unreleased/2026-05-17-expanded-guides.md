date: 2026-05-17

The `?` Guide hub now covers the features that landed in v0.5.0.

- Two new guides — **Mock server** and **Response viewers** — walk
  through serving saved examples over HTTP and previewing binary
  responses.
- The **Auth** guide gained AWS Signature v4 and mTLS sections.
- The **Body modes** guide now covers multipart/form-data and
  binary file uploads.

The GitHub README's feature list is caught up to match.
