# Add more languages (DE, ES, FR, JA, ZH, AR)

GitHub Issue: [#14](https://github.com/olgunozoktas/api-lab/issues/14)

i18n is in place (TR + EN). Adding a new language is mechanical:

1. Create `frontend/src/lib/i18n/<code>.ts` exporting a `Dict`-typed object (TS will fail the build if a key is missing)
2. Register in `frontend/src/lib/i18n/index.ts` `locales` map and `LOCALE_LABEL`
3. Append to `SUPPORTED_LOCALES`
4. Add the new label key (`lang.<code>`) to all existing locales

Priority languages: Spanish (ES), German (DE), French (FR), Japanese (JA), Simplified Chinese (ZH), Arabic (AR — also enables RTL audit).
