# Phase H — Distribution

GitHub Issue: [#8](https://github.com/olgunozoktas/api-lab/issues/8)

End-user installable, no compile required.

- macOS: `.dmg` via `create-dmg` from CI; sign + notarize (requires Apple Developer cert — separate ask)
- Linux: AppImage (single binary, runs anywhere), `.deb` for apt, optional flatpak
- Windows: signed `.exe` installer (innosetup or wix), optional `winget` manifest
- Brew formula (homebrew tap or homebrew-core PR — start with personal tap)
- Auto-update notifications: GitHub Releases API check on launch
- Release notes auto-generated from CHANGELOG.md
