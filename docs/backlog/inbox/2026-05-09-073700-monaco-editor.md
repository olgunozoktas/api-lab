# Replace body/headers textareas with Monaco editor

Power users want a real code editor for body (JSON, GraphQL, raw) and pre-request scripts (Phase I). Monaco is what VS Code uses; integrates well via `@monaco-editor/react`.

- `dnpm install @monaco-editor/react monaco-editor`
- Lazy-load (Suspense + dynamic import); keep first paint fast
- Bundle impact: +2-3 MB; keep behind a feature flag or only load on body tab open
- Themes: tokyo-night, github-light, vs-dark — match app theme
- JSON schema validation when `Content-Type: application/json`
- GraphQL highlight in graphql panel
- Keyboard shortcuts integration (preserve our ⌘+Enter / ⌘+S overrides)
