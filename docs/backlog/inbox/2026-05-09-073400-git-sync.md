# Phase K — Optional git-based sync

Bruno-style. User picks a git repo; collections + envs sync as plain JSON files. No vendor lock-in, no cloud account. Opt-in only.

- Settings: "Sync repo URL" + auth method (SSH key path or PAT)
- On save: write to local clone, `git add` + `git commit` + `git push` (background)
- On launch: `git pull` if configured
- Conflict resolution: surface manual diff, no auto-merge
