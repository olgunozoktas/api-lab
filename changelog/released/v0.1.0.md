API Lab — a native macOS API tester (Postman-style) built on the
zero-native Zig + WebKit shell. CORS-free HTTP via curl subprocess,
WebSocket / SSE / gRPC tabs, request history, environments,
collections, GraphQL composer, multi-tab workspace, TLS config,
Postman import.

Highlights from this first release:

- **Multi-protocol composer** — HTTP, GraphQL, WebSocket, SSE, gRPC
  (unary + reflection-cached service browser) all in one window.
- **Request cancellation** — Send button morphs to Cancel; ⌘+. fires
  the same handler. AbortSignal threaded through the fetch path.
- **gRPC TLS** — per-request tmpfile credentials with hardened
  permissions (0o700 dir / 0o600 PEMs).
- **Environment variables** — `{{var}}` substitution across URL,
  headers, body, query params. Per-environment vars + active
  environment switcher in the top bar.
- **Collections + history** — folder-tree organization, inline
  rename + drag-to-move, request snapshots in history with elapsed
  time + status code badges.
- **Examples** — saved-response captures keyed off saved requests.
- **Multi-tab workspace** — Cmd+T new, Cmd+W close, Cmd+1..9 switch,
  Cmd+P quick switcher across collections + tabs.
- **Postman import** — drag-drop a `.json` collection to import the
  whole tree + env vars in one shot.
- **Themes + Turkish/English** — system / light / dark; full TR + EN
  i18n.
