API Lab — a native macOS API tester (Postman-style) built on the
zero-native Zig + WebKit shell. CORS-free HTTP via curl subprocess,
WebSocket / SSE / gRPC tabs, request history, environments,
collections, GraphQL composer, multi-tab workspace, TLS config,
Postman import.

This entry backfills the major features that landed during the v0.1.0
build-up. From v0.1.1 forward, every user-visible change ships as its
own `changelog/unreleased/<slug>.md` file.

## Composer + protocols

- **HTTP composer** — method picker, URL bar with `{{var}}` substitution,
  Params / Headers / Auth / Body sub-tabs, raw body modes (json / form-
  urlencoded / raw / none), headers KV editor with non-ASCII guard.
- **GraphQL composer** — auto-detected when the URL points at a GraphQL
  endpoint; method becomes POST, body serialized as `application/json`.
  Query + variables in dedicated panes.
- **WebSocket panel** — `ws://` and `wss://` URLs activate the WS tab.
  Connect / disconnect, send messages with `⌘ Enter`, ping button,
  per-connection log with auto JSON detection.
- **Server-Sent Events panel** — `sse://` and `sses://` URLs activate
  the SSE tab. Auto-reconnect handling, named-event tags, last-event-id
  cursor capture.
- **gRPC panel** — `grpc://` and `grpcs://` URLs activate the gRPC tab.
  Routes through `grpcurl` (must be installed). Unary + server-streaming
  via batched-collect approach.
- **gRPC reflection service browser** — when the server has reflection
  enabled, list services + methods in a sidebar; pick a method to get a
  request-message skeleton. Per-target 5-min cache with `(cached Xs ago)`
  badge on hits; manual Refresh button invalidates.
- **gRPC TLS / mTLS** — paste-in CA cert, client cert, client key (PEM)
  with per-call temp files. Hardened permissions: `0o700` for the dir,
  `0o600` for each file. SNI + authority overrides supported.

## Workspace + organization

- **Collections** — folder-tree sidebar with inline rename, drag-to-move,
  recursive delete, and a context menu (right-click) for new sub-folder /
  rename / delete. Search filters visible nodes live.
- **History** — every request lands in history with method + status +
  size + elapsed-ms badges. Smart search across method + status + URL
  substring (e.g. type `GET 401 login`).
- **Multi-tab workspace** — `⌘ T` new, `⌘ W` close, `⌘ 1`–`⌘ 9` jump.
  Each tab carries its own request body / response state; collections /
  environments / history are shared.
- **Quick switcher** — `⌘ P` opens a fuzzy palette across open tabs +
  saved collections + recent history. Smart substring scorer.
- **Postman v2.1 import** — drag-drop a collection JSON to bring the
  whole tree + collection variables over in one shot. Imports land
  under a wrapper folder named after the source.
- **Resizable 3-pane layout** — drag dividers to resize Sidebar /
  Composer / Response. Double-click to reset. Widths persist across
  reloads.

## Authoring + automation

- **Environments** — multiple named contexts, each with its own KV var
  set. Active env in the top-bar dropdown; `{{var}}` substitution
  happens at send-time across URL / headers / params / body.
- **Save as variable** — right-click any value in the response viewer
  to save it into a named env variable (`{{access_token}}`, etc.) for
  use in later requests.
- **Examples** — captured response snapshots attached to a saved
  request. Save / view / rename / delete from the Examples tab. Persist
  with the request so they survive reloads.
- **Pre / post-request scripts** — QuickJS sandbox with a `pm.*` API
  subset (pm.test, pm.expect, pm.environment.set, etc.). Pre-scripts
  can mutate the request before send; post-scripts run on the response.
  Scripts panel surfaces test results + console output inline.

## Cancellation + observability

- **Request cancellation** — Send button morphs into a red `⨯ Cancel`
  while in flight. `⌘ +.` (canonical macOS abort) fires the same
  handler. AbortSignal threaded through the fetch path; native bridge
  path returns UI to ready immediately, with documented soft-cancel
  semantics for the underlying curl subprocess.
- **Aria-live response status** — screen readers announce status code
  changes when a request lands.
- **Astro-style error overlay** — developer-side: when the React tree
  crashes, a syntax-highlighted blame frame + parsed component stack +
  source excerpts surface in-app instead of a blank screen.

## Look + feel

- **Themes** — system / light / dark / Tokyo Night / GitHub Light /
  high-contrast. Auto follows the OS appearance unless overridden.
- **Turkish + English i18n** — every user-facing string flows through
  `t()`. TR is the source of truth; EN mirrors it. Adding a new
  language is a single dictionary file.
- **Keyboard shortcuts** — full reference in the Settings modal:
  `⌘ Enter` send, `⌘ S` save, `⌘ N` reset, `⌘ T` new tab, `⌘ W` close
  tab, `⌘ 1`–`⌘ 9` jump, `⌘ P` quick switcher.

## Platform

- **Native macOS shell** — built on zero-native (Zig + WKWebView).
  CORS-free HTTP via curl subprocess routed through the Zig bridge;
  the rest (UI, persistence, history) lives in the React frontend.
- **IDB persistence** — collections, environments, history, tabs, and
  preferences live in IndexedDB. Uncaps the old localStorage 5 MB
  limit. Persist contract is versioned (v1 → v2 → v3 migrations);
  user state survives across upgrades.
