---
title: Right-click a folder to add a new request inside it
date: 2026-05-10
---

The folder context menu in the collections sidebar gained a **New
request** entry above **New sub-folder**. Right-click any folder, pick
**New request**, and an empty request lands directly under that folder
+ becomes the active request in your composer — ready to edit.

The folder auto-expands so the new entry is visible without an extra
click.

Same flow as the existing **+ New Request** button at the bottom of
the sidebar, except the latter creates an unsaved blank in the active
tab; this one persists into the targeted folder immediately so it
survives a reload without an explicit save step.
